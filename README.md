## 📖 Описание программы

**Training Planner** - это графическое приложение для планирования и отслеживания тренировок. Программа позволяет:

- ✅ Добавлять тренировки с указанием даты, типа и длительности
- ✅ Просматривать все тренировки в удобной таблице
- ✅ Фильтровать тренировки по типу или дате
- ✅ Удалять ненужные записи
- ✅ Просматривать статистику тренировок
- ✅ Сохранять данные в JSON файл и загружать при запуске
- ✅ Проверять корректность ввода данных
